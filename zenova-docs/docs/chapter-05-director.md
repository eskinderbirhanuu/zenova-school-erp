## 🎯 ማጠቃለያ (Summary)



ዳይሬክተሩ የትምህርት ቤቱ ዋና አካዳሚክ እና አስተዳደር ተቆጣጣሪ ነው። የተማሪ መገኘት፣ የመምህራን አፈጻጸም፣ የፈተና አስተዳደር እና ለባለቤቱ ሪፖርት ማቅረብ ዋና ዋና ተግባራቱ ናቸው።
