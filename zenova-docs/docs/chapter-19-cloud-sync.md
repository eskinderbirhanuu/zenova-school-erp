## 🔄 የማመሳሰል ሂደት (Sync Process)



```mermaid

sequenceDiagram

    participant School as 🏫 የት/ቤት አገልጋይ

    loop በየ30 ደቂቃው

        School->>Sync: ለማመሳሰል ዝግጁ

    end
```
