# Glossary

| Term | Definition |
|------|------------|
| **Branch** | Sub-school instance under a main school |
| **Chapa** | Ethiopian payment gateway (telebirr, CBE, etc.) |
| **God Node** | Auto-generated graphify node summarizing an entire file |
| **JWT** | JSON Web Token — stateless auth via RS256 or HS256 |
| **Main School** | Top-level school entity; owns branches |
| **NFC** | Near-Field Communication — used for student ID cards |
| **Registrar** | Role that manages student enrollment and records |
| **Server Identity** | On-disk JSON (`server_id.json`) identifying this Zenova instance |
| **SSRF** | Server-Side Request Forgery — prevented by URL validation + DNS rebinding checks |
| **Sync Queue** | DB table that queues outbound changes for VPS replication |
| **Telebirr** | Ethiopian mobile-money service (Chapa gateway) |
| **VPS** | Virtual Private Server — remote sync target for multi-site deployments |
| **Watermark** | School/branch identifier embedded in exported PDFs and images |
| **Zenova** | School management platform (this project) |
